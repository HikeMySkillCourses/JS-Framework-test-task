### Краткое описание
Разработать систему тестирования. Пользователь может просматривать задания, смотреть отдельное задание, сабмитить, смотреть навыки
Пример для всего можно смотреть на [сайте](https://hikemyskill.com/ru)

Вспомагательные файлы для Mock Api лежат в папке `data`

### Требования к страницам
- Страницу показа навыков (`skill-tree.json`)
    - Навыки в виде дерева
    - Должны открываться по нажатию
- Страницу со всеми заданиями (`get-all-tasks.json`)
    - Сделать фильтр по Сложности
    - Если для задания поле `isPassed = true` -> помечать как выполненное
- Страница одного задания (`single-tasks/{номер}.json`)
    - Просмотр, выполнение
      - К каждому вопросу присылается навык - нужно группировать их и показывать **только** уникальные значения  
      - Если текст обёрнут в `/``/` - применять к нему доп стили 
      - Если для задания приходит `result` -> показывать задание как выполненное
    - После выполнения переброс на страницу всех заданий
